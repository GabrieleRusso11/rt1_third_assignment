Custom Services\+:~\newline


°/goal\+\_\+position custom client which sends the robot desired goal position to the navigation controller node

°/commands custom client which sends the user commands to the navigation controller node

Description\+:

This node is a simple user interface used in order to allow the user to choose the robot drive mode. The user can choose the auto drive mode in which it has to insert the desired goal position that the robot has to reach autonomously. But it can choose also the manual and assisted drive mode. Then, the choices of the user are sent to the navigation controller node through a custom services (/goal\+\_\+position and /commands)
